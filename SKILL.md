## üìê MODULE 5: POST STRUCTURE BLUEPRINTS (8 Types)

**1. THE STORY ARC** (1000-1500 chars)
*   **Skeleton**: In Media Res -> Context -> Tension -> **Pivot** -> Resolution -> Lesson -> Conversation Invitation.
*   *Best for*: Vulnerability, Milestones.

**2. THE LISTICLE WITH TEETH** (800-1200 chars)
*   **Skeleton**: Bold Hook -> Context -> Items 1-X (Punchy) -> **Unexpected Final Item** -> Closer.
*   *Best for*: Tactical Value, Saves.

**3. THE HOT TAKE** (600-1000 chars)
*   **Skeleton**: Bold Claim -> Acknowledge Crazy -> Evidence -> Reframe -> Nuance ("I'm not saying...") -> Debate Invite.
*   *Best for*: Reach, Comments.

**4. THE BEFORE/AFTER TRANSFORMATION** (800-1200 chars)
*   **Skeleton**: Painful Before (Sensory) -> Breaking Point -> The Shift (Specifics) -> After Results (#s) -> Real Lesson -> Invitation.
*   *Best for*: Credibility.

**5. THE FRAMEWORK DROP** (900-1300 chars)
*   **Skeleton**: Hook (Problem) -> Credibility -> Steps 1-5 (Actionable) -> Application -> Save Trigger.
*   *Best for*: Educational Authority.

**6. THE DATA PLAY** (600-900 chars)
*   **Skeleton**: Surprising Stat -> Context -> Analysis -> So-What -> Action -> Question.
*   *Best for*: Industry Commentary.

**7. THE MICRO-STORY** (400-700 chars)
*   **Skeleton**: Cinematic Moment (Present Tense) -> Pause (White Space) -> Meaning -> Universal Truth.
*   *Best for*: Emotional Connection.

**8. THE QUESTION POST** (400-600 chars)
*   **Skeleton**: Provocative Question -> Your Perspective -> The Twist -> "I want to hear your take."
*   *Best for*: Community Engagement.
